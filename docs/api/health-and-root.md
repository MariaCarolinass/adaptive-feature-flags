# Root e Health

## `GET /`

Retorna uma mensagem simples de identificação da API.

Exemplo de resposta:

```json
{
  "message": "Adaptive Feature Flags"
}
```

## `GET /health`

Endpoint de healthcheck para monitoramento básico de disponibilidade.

Exemplo de resposta:

```json
{
  "status": "ok"
}
```
